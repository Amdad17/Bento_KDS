Rule Descriptions

Rider Distance

When an order comes in and a rider is said to arrive at the restaurant in the time set in this rule or before, then the order will get importance over others.

Customer Wait

When an in-house order has been waiting for the time given in this rule, it will be given priority.

Course Wait

When an in-house order has multiple courses and a course has been waiting for the time given in this rule, it will be given priority.
